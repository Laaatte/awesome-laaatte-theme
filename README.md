# Awesome Laaatte Theme

개인적으로 쓰려고 만든 VS Code 라이트 테마.  
화이트 배경에 코드 요소별 색상을 확실히 구분해서 눈 덜 피로하게.

## 색상 구성

| 요소 | 계열 |
|------|------|
| 배경 | 그레이 화이트 |
| 키워드 | 파랑 |
| 문자열 | 그린 |
| 함수 / 메서드 | 크림슨 |
| 클래스 / 타입 | 딥 퍼플 |
| 변수 | 번트 오렌지 |
| 파라미터 | 포레스트 그린 |
| 숫자 | 앰버 |
| 연산자 | 미디엄 퍼플 |
| 주석 | 그레이 |

## 설치

1. `Cmd+Shift+X` → `Awesome Laaatte Theme` 검색
2. 설치 후 `Cmd+Shift+P` → **Color Theme** → **Awesome Laaatte Light** 선택
